Esta es la parte 2 de la actividad 1

INT: En esta variable solo se pueden especificar numero, pero no cualquier numero, si no numeros enteros

CHAR: En esta variable se especifican letras o numeros, no mas de uno porque eso se haria en la variable string string

STRING: En esta variable se pueden escribir palabras, ya que esta variable es como una cadena de varios char

FLOAT: En esta variable se pueden colocar numeros decimales

BOOL:En esta variable solo hay 2 unicos valores, en esta variable los valores son en binario: TRUE o FALSE, falso se muestra con el numero 0 y verdadero con el 1

Si quieres guardar el resultado de algo debes crear otra variable, pero si no puedes imprimir solamente lo que deseas, y el programa lo borrara.

CONDICIONALES:

Hay 2 caminos en el condicional, dependiendo si se cumple o no se cumple, si se cumple va por un caminio, y mostrara una cosa, si no se cumple el condicional ira por el otro y mostrara otra cosa
